# Find Your Pizza

***Find your pizza***는 피자에 MBTI유형을 비유한 성격유형검사 서비스입니다.<br/>
간단한 성격 유형 검사를 통해 자신의 MBTI와 비슷한 느낌의 피자를 매칭시킨 결과를 확인할 수 있습니다.
